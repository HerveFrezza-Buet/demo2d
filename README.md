# Unix Installation

First, get the files.

``` 
git clone https://github.com/HerveFrezza-Buet/demo2d
``` 

Then, you can install the two packages as follows.

```
mkdir -p demo2d/build
cd demo2d/build
cmake .. -DCMAKE_INSTALL_PREFIX=/usr
sudo make install
cd ../..
```

The documentation is in /usr/share/doc/demo2d
